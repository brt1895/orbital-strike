# Orbital Strike

Gunes sistemi temali, tarayicida calisan 2D yercekimi atis oyunu.

## Ozellikler

- Dunya uzerindeki firlatma istasyonundan meteor atisi
- Gunes ve gezegenlerin cekimiyle buken rota fizigi
- 7 seviyeli hedef sistemi
- Mobil uyumlu arayuz
- PWA destegi ile uygulama gibi kurulabilme
- GitHub Pages ile yayinlanmaya hazir statik yapi

## Yerel Calistirma

Bu proje sadece statik dosyalardan olusur. `index.html` dosyasini bir tarayicida acabilir veya basit bir HTTP sunucusuyla calistirabilirsin.

## Yayinlama

`.github/workflows/deploy-pages.yml` dosyasi sayesinde repo `main` branch'ine push edildiginde GitHub Pages deployment akisi tetiklenir.
